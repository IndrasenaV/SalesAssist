Development setup
==============================

1) Download Github
2) Download Sencha cmd 5.0
3) Setup up cordova article is here http://docs.sencha.com/touch/2.3.2/#!/guide/cordova
4) Download node.js from here http://nodejs.org/download/



Sencha commands

Sencha Cmd Version : 5.0

Sencha light weight web container

sencha web start

for testing
===============

sencha app build testing

for compressed production
========================

sencha app build production

for native app generation
============================

sencha app package build <configFile.json>
